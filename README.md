# pac-man
Vanilla javaScript Game with HTML and CSS

A super simple walkthourgh to coding Pac-Man in vanilla JavaScript, HTML and CSS.

### Inbuilt JavaScript functions I have used: 
* addEventListener()
* getElementById()
* querySelector()
* for Loop
* appendChild()
* createElement()
* push()
* if else statements
* classList
* switch case
* keyCode
* forEach()
* setTimeout
* Class
* constructor
* setInterval()
* removeEventListener
